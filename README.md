KalaSetu – AI-Powered Handmade Marketplace

Overview

KalaSetu is an Android application designed to connect local artisans with customers through a modern digital marketplace. The platform enables artisans to showcase handmade products, manage orders, and track sales while providing customers with a seamless shopping experience.

Features

Customer Module

- User Authentication
- Product Browsing
- Product Search
- Product Details View
- Wishlist Management
- Product Reviews & Ratings
- Notifications
- Customer Profile
- Order Tracking

Artisan Module

- Artisan Dashboard
- Add Product
- Product Management
- Order Management
- Sales Analytics
- Artisan Profile

Admin Module

- Admin Dashboard
- Product Monitoring
- Order Monitoring
- Artisan Monitoring
- Reports Management

---

Technologies Used

- Kotlin
- XML
- Android Studio
- Firebase Authentication
- Firebase Firestore
- RecyclerView
- CardView
- Material Design Components

---

Project Architecture

KalaSetu follows a role-based architecture consisting of:

1. Customer Panel
2. Artisan Panel
3. Admin Panel

Each module has dedicated screens, navigation flows, and Firebase integration.
